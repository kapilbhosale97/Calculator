# Calculator Project

A simple GUI calculator built with Python and Tkinter.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Percentage calculation
- Sign toggle (±)
- Clear function
- Error handling

## Requirements

- Python 3.x
- Tkinter (included with Python)

## How to Run

```bash
python calculator.py
```

## Usage

- Click numbers and operators to build expressions
- Press '=' to calculate result
- Press 'C' to clear
- Press '±' to toggle sign
- Press '%' for percentage calculation
